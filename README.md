## Develop

```
git clone https://github.com/m4xshen/CanChu.git
cd CanChu/
yarn install
yarn run dev
```

## Docker

1. Build the container: `docker build -t nextjs-docker .`
2. Run the container: `docker run -p 3000:3000 nextjs-docker`
